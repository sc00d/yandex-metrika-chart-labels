## Установка

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для своего браузера

2. Нажмите на иконку Tampermonkey → **Создать новый скрипт**

3. Удалите весь дефолтный код и вставьте содержимое файла [`ym-labels`](./ym-labels.js)

4. Нажмите **Ctrl+S** 

5. Откройте Метрику — метки появятся автоматически ✅

## Возможности

- **Автозапуск** на всех страницах `metrika.yandex.ru` и `metrika.yandex.com`
- **Автообновление** меток при смене периода, метрики или фильтров — через `MutationObserver` с дебаунсом 700 мс
- **Все линии** — работает для графиков с несколькими метриками одновременно
- **Точные числа** с форматированием `13 045` через `toLocaleString('ru-RU')`
- **Округление** — опциональный режим: `13,0 К` / `1,2 М`
- **Умное размещение** — если подписи накладываются, алгоритм автоматически разносит их по высоте и добавляет пунктирную линию-связь
- **Стильные точки** — белый круг с цветной обводкой, цвет берётся с линии графика
- **Подписи** — цветной прямоугольник с белым текстом, ширина подстраивается под число
- **Все значения оси Х** — выводятся все месяцы на шкале
